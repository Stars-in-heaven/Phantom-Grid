// What Is Phantom Grid

Built for
the Field.

Phantom Grid is a series of purpose-built handheld devices designed for those who need local intelligence without a network. No Wi-Fi. No cloud sync. No IP assignment. No detectable radio signature.

When you want to load files onto the SD card, you physically remove it, write to it externally, and return it. That's not a limitation — that's the point.

"Inconvenience is the mechanism by which Phantom Grid remains undetectable, untrackable, and uncompromised."

// Current Build

Phantom Grid T-V.1

01
Tri-Display Architecture
Three independent displays — 0.91" OLED for status, 1.5" OLED for local data, 2.4" TFT colour terminal for file browsing.
02
Dual ESP32-C6
One chip per side. Each side independently operable. Connected via UART over a 4-wire hinge cable. One side compromised does not expose the other.
03
Air-Gapped Storage
64GB microSD loaded by physical removal and external write. No cloud sync, no Wi-Fi, no IP assignment required.
04
Field Sensors
GPS module, temperature sensor, and moisture sensor all on the left side. Local intelligence without a network.
05
Firmware Hardening
CRC self-check on boot, chip ID binding, control flow obfuscation, runtime-derived sensitive logic. Binary-only distribution.
06
4–6hr Battery Life
2000–2500mAh LiPo with TP4056 dual protection charging. Left side owns power and feeds right side via hinge rail.
